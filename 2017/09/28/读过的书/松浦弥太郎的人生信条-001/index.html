松浦弥太郎的人生信条-001

发表于 2017-09-28 更新于 2018-08-13 分类于 读书 阅读次数： 本文字数： 2.9k 阅读时长 ≈ 3 分钟

猫空看到的一本书，看了几页试读之后，决定买下来慢慢看。
需要不断的领悟，反复的参透。

我的得到时光书签也有了一个好的归宿。

001 一切自己负责。指责他人也无济于事。

好事，坏事，我们会经历各种事情。无论发生什么事，我认为“一切的起因都在自己”。不推到他人，社会的身上，不去指责他人，也不怨恨这社会。不管发生什么，自己处理，自己主动承担责任。大体的事情，都可以自己负责解决。只有带着”说服自己的关键是在于自己“的觉悟，才可能不依存于他人，用自己的双脚走下去。

经过这么多次的实践教育，我知道抱怨，指责是无济于事的。如果抱怨，指责能够让对方做出改变，那么和颜悦色的去交流的效果会更好。
要认识到人与人之间的差异性，要理解那些不理解的事情，是因为你没有站在对方的角度，用对方的眼光去看待。
虽然不知道后面的99条是什么，但是这第一条应该是最重要的一条。也是我们每个人都应该做到的一条。

002 舍弃自尊，凡事忍耐。

自尊心，不是用来肆意宣扬的武器。将自尊深藏于内心就好。舍弃自尊，凡事忍耐。这两条，是我所尊敬的一位名编辑交给我的，让工作得以长久的秘诀。“动不动就暴怒的话，工作就无法开展。请舍弃自尊，即便觉得不对也学会忍耐。保持冷静思考。”这是我从他身上学到的珍贵哲理。

让我哭一会儿吧，这两条我都犯了，特别是第二条，我觉得自己的自尊心太强了，现在我最大的问题就是如何面对那些不可理喻的人。
和他们沟通，太累了。性格比较急吧，我看慢性子到是没问题，什么都慢吞吞的，像蜗牛一样。
道理都知道，但是还是无法忍受。
慢慢努力吧，偶尔也去慢慢的接触一下不可理喻的人，增加一下自己的耐受度。

003 简单地生活。要做的事，保持一点点就好

简单思考后，选择少量“应该做的事”。如果其中有复杂的事就简单化，认认真真地拼命做到最好。如果什么都插一手，最后到结果不过是每件事都半途而废罢了。“想穿红色的，也想穿蓝色的”，两种颜色都上身，只会显得不伦不类。只要有”想要那样，有想要这样“的想法，就不可能全部实现。不能明确目标，凡事都只能是浅尝辄止。

这条也是金玉良言，似乎做到这前面的3条就很牛了。鞭策自己。

004 不对过去说谎

大家都知道不该说谎，但那只是对当下而言。对于过去，是不是会为自己编排上更有利的情节呢？即便不算说谎，也会为了将自己现在的行为正当化，而稍稍调整过去的故事吧？过去的事，谁也没法去验证。小小的谎言也不会被揭露。但正因如此，才罪恶深重。所以我认为，最不该说的，就是对“过去”撒的谎。

对这条的理解还不是很深刻，可能是说在有能力去做坏事的时候，而选择不做就是一种善吧，更是对自我的一种把控。

005 做被钱喜欢的工作，过被钱喜欢的生活

己所不欲，勿施于人。对人对钱都是如此。对工作和生活来说，钱都很重要。钱是经常帮助自己，像朋友般的存在。所以每次打开钱包时，都问问自己的内心吧。“把钱用在这上面，她是会讨厌，还是高兴”。以此为基准，摸索被钱喜欢的工作方式，被钱喜欢的生活方式。

应该是说选择行业的事吧。

006 最重要的工作，是规律生活和健康管理。

我从事文字工作，经营旧书店“COW BOOKS”，担任《生活手帖》的主编。在所有事情中，排在第一位的，是自己的健康管理。第二位，是保护为我工作的社员们的健康。过规律的生活。不加班，也不让别人加班。如果发现有工作多到要睡不着觉的人，就会伸出援手。身体不健康也做不了好的工作，这是适用于所有人的原则。

是的，身体是革命的本钱。

007 无论做什么事，都要想到下一个人。

上厕所的时候，要想到下一个用的人。扔垃圾的时候，要想到将垃圾送到垃圾场的人，回收运输的人，处理垃圾的人。做杂志的工作，要想到校稿的人，印刷的人，装订的人，将书运到书店的人，为我们卖书的人，以及读者。无论做什么事，都不要忘记人的存在。我想牢记这一点做事。

这个我的体会很深，可能因为我是个敏感的人吧，我总能看到那些仅仅是举手之劳但是却不考虑后面的人的情况。我相信每个人都希望生活在一个美好的环境当中，而这个美好的环境，恰恰是我们自己营造的。近朱者赤，近墨者黑。想想自己做的事情，能不能让自己生活在一个美好的环境中呢。

008 抢占先机，确认工序，准备充分，仔细作业

工作基本上都是准备。准备充分了基本上都会达成。抢占先机，确认工序，只要有过仔细的准备。基本上事情都会顺利进行。稍微有点困难的，是抢占先机的时间点。出手太迟，就无法顺势而为；出手太早，也会出现问题。这方面经验的积累固然重要，但要有这种意识；想到“差不多必须开始了”的时候，通常都已经太迟了。

做任何事情基本上都是准备，这个我是认同的，准备充分和抢占先机似乎是矛盾的，需要寻找之间的平衡点。看这条结合目前社会上公司的状态似乎都在做所谓的抢占先机，推出问题多多的产品，然后不断地打补丁。似乎运行的也还可以。真是无法判断好与坏了。

009 沟通，就是为了传达爱愿

无论是工作还是生活场合，与人沟通的能力都很必要。做演讲也好，向人说明事理也好，包括和重要的人加深关系，交流都不可或缺。正因为如此，一定要明白这点：交流的目的，是为传达爱愿。对工作的爱愿，对人的爱愿，对物的爱愿，对项目的爱愿。不要忘记交流的目的是传达爱愿。

嗯，从来没有这么考虑过，沟通是无法避免的，学习中。

010 幸福是人与人之间深刻的联结和深深的羁绊

“对你来说，幸福是什么”，如果被这样问你能迅速作答吗？我们都是为了获得幸福而生存。为他人和自己的幸福而工作，生活。知道自己的幸福为何，也就等于知道了自己在追求着什么生存。对我来说，幸福就是和人深深的联结。加深与人的羁绊，是我最大的幸福。因为能够看到前方那”幸福的景色“，我才得以每天都拼尽全力。

我还很年轻，还不知道幸福是什么。 突然想起一个网络段子。

011 不破坏，不失信，不汲汲以求，不放弃

对工作，对生活，对任何事都是如此。不破坏与人之间的联结关系。不破坏约定。不管做什么，都能够持之以恒地努力。一旦开始，不放弃就很重要。无论如何注意，总有搞砸，失信的事情发生。正因为基于这样的事实，才要为追求不破坏，不失信，不汲汲以求，不放弃而努力。

我一直在不破坏，不失信，这个是我的做人宗旨。 不汲汲以求也算是可以做到吧。不放弃，这一点我似乎还做得很不够。我想要做个乐观主义者，要锻炼自己从乐观的角度看待任何事情。

012 栽培，守护，坚持

人与人之间的关系，因培育而生。建立起关系后，就好好守护吧，使之持续下去。“如何才能栽培？如何才能守护？如何才能坚持？”要对这些问题做多方面的思考。

有道理。

013 愈是小的承诺愈要信守

“下次，一起吃饭吧” “那本书我借你”，平时这些话我们经常脱口而出。可能都是顺着话题随口说的，但这和郑重的约定，重大的约定一样，都应该被遵守。愈是不知不觉间许下的模糊不清的诺言，愈是小小的约定，愈重要。不轻视小约定，然后去执行，对方会欣喜地感到”啊，他还记得啊” “原来那不是一句客套”。我想传达给对方种喜悦。

很欣慰，终于有一条是我一直信守的。

014 放大镜和望远镜

要学会使用放大镜和望远镜。看清楚近物和远物的能力合在一起，才能更接近事物的本质。对周身的事物和世界形势都是如此。在自身的时间轴上，“现在的事和二十年后的事”，也是我想用放大镜和望远镜两个视点考虑的事情。

有道理。需要学习，提醒自己。
