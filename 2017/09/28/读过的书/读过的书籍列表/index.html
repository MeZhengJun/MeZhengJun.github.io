发表于 2017-09-28 更新于 2022-12-12 分类于 读书 阅读次数： 本文字数： 5.9k 阅读时长 ≈ 5 分钟

2013 年 3 月份来到上海，最常去的地方有三个，按时间长短排序：公司，家，图书馆。
喜欢上海图书馆总馆种类的繁多，畏惧餐厅的午餐+晚餐。
喜欢浦东图书馆宽敞大气的学习室+美味的午餐和晚餐，还有咖啡，糕点，真是太棒了。美中不足的就是 WIFI 好难连接。
听说嘉定图书馆是美国权威设计杂志评选的 2013 年“全球最佳公共图书馆”，于是去欣赏了一下，大自然是均衡的，美有了，于是内容就没了。
徐汇图书馆小小的，却有着一些意想不到的收获。
