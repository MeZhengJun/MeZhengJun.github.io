精油疗法-配方

Posted on 2017-12-23 | In 健康

集中注意力

迷迭香1滴 + 薰衣草1滴 + 薄荷1滴 + 尤加利1滴

漱口

茶树精油1滴，薄荷精油1滴，水500ml。
精油混合后加入温水，摇匀漱口即可。

去除异味
罗勒精油1滴+温水

没药治疗所有的牙齿感染，对喉咙痛，口腔发炎，牙龈肿痛都有舒缓效果

没药1滴+迷迭香1滴+温水一杯

口气芬芳，解决口臭

柠檬精油5滴+薄荷精油2滴

丁香精油1滴+300ml水，摇晃均匀，正常漱口
经常用来治疗牙痛

马鞭草
缓解牙龈肿痛，扁桃体发炎

放松肌肉

马郁兰精油4滴+迷迭香精油4滴+柠檬草精油2滴+葡萄籽油10ml+甜杏仁油10ml
改善肌肉疼痛，关节痛等症状有很好的效果

丁香3滴+雪松3滴
丁香是运动员的好伙伴，对运动扭伤，关节受伤，可以缓解疼痛

肩颈酸痛

荷荷巴油针对背部疼痛，吸收效果好。
薰衣草精油：殺菌、消毒，性質溫和，味道可鎮定安神。
檸檬尤加利精油：促進新陳代謝，與薰衣草精油混合後有效消除疲勞，令身體毛孔張開，釋放壓力。
黑胡椒精油：令頭腦醒神，恢復活力。
适用精油：迷迭香、薰衣草、桦木、马郁兰、刺柏莓。

马郁兰，罗马洋甘菊，纯正薰衣草

一种缓解颈肩酸痛的复方精油专利配方，由以下植物精油按体积份数组成，马郁兰精油10~20份、生姜1~5份、罗马甘菊精油5~10份、薄荷精油1~3份、佛手柑精油1~3份、甜杏仁油200~600份

马郁兰精油：对肌肉疼痛、痛经、头痛、消化系统疾病等都有很好的疗效。治疗哮喘和其他呼吸道疾病方面很有价值，对情绪有镇静作用。它能帮助活动亢进的人平静下来，还可以促进循环。马郁兰精油是心脏的补药，能降低高血压。它的放松效果有助于改善头痛、偏头痛和失眠。马郁兰精油在安抚消化系统的效果相当闻名，有益于胃痉挛、消化不良、便秘、胀气，还能帮助身体排除毒素，以及预防晕车、晕船。似乎有益于胸腔的传染病及感冒、鼻窦炎、支气管炎与气喘。在感冒时，能消除头脑的窒塞感。调节月经周期，减轻经痛。它抑制性欲的作用也十分着名。 消除淤血的作用极富价值，它能消除淤血，是因为扩张微血管后血液轻易流通
与马郁兰精油调配的精油
佛手柑精油、迷迭香精油、花梨木精油、依兰精油、熏衣草精油、雪松精油、洋甘菊精油、丝柏精油、桔精油、橙精油、肉豆蔻精油。
提示
孕妇禁用。马郁兰精油使用时间过长可能会导致精神状态迟缓，气喘者、低血压者、忧郁症者小心使用。

罗马洋甘菊：肌肉酸痛和神经性疼痛，极佳抗痉挛与镇静消炎作用
快乐鼠尾草：乳酸代谢，缓解肌肉紧绷
纯正薰衣草：缓解肌肉僵硬和紧绷
小花茉莉：缓解肌肉僵硬和疼痛
黑胡椒：肌肉酸痛，僵硬与疲劳，训练增进肌耐力，减少过度耗损疼痛

缓解肩颈疼痛的精油：檸檬尤加利精油，杜松精油、葡萄籽油

缓解肩颈压力的精油：主要配方是薰衣草精油、迷迭香精油、橄榄油

25滴精油 + 30ml调和油：
罗马洋甘菊6+马郁兰10+姜4+薰衣草4+柠檬尤加利2+佛手柑2

失眠

橘子精油2滴，罗勒精油3滴，薰衣草精油3滴：香薰
强化神经系统，对抗神经疲劳，神经性失眠等问题

过于疲惫急于放松

乳香精油 2滴 + 葡萄柚精油2滴 + 甜橙精油1滴

孤独，寂寞，紧张，亢奋

马郁兰精油3滴+薰衣草精油3滴+橙花精油1滴

草莓鼻

用葡萄籽油10ml（质地较清爽、易吸收）薰衣草3滴、葡萄柚1滴、杜松1滴。
薰衣草可使皮肤正常进行新陈代谢，预防角质堆积肥厚；
佛手柑则有净化毛孔，修护发炎肿胀毛孔的效果；
杜松对付毛孔粗大、粉刺肤质更有杀菌与淋巴排毒的卓越效果。

青春痘

薰衣草精油滴在棉棒上，直接涂抹在青春痘上，可以消炎，帮助愈合
薰衣草精油是少数可以直接在皮肤上使用的精油，但是也要注意个体的差异，避免过敏

消毒杀菌

牛至精油

睡眠

薰衣草2滴 + 快乐鼠尾草3滴 + 乳香3滴
使用方法：睡前熏香，帮助睡眠，安抚激动情绪

腹泻，痢疾

薄荷精油3滴 + 洋甘菊2滴 + 尤加利1滴 + 芝麻油5ml
使用方法：调和后涂抹于肛门处，可迅速止泻

胃胀，反胃

豆蔻精油3滴，姜精油1滴，橄榄油10ml
混合后按摩胃部，腹部即可改善胃肠状况

腰酸

10滴荷荷巴油 + 冬青2滴 + 杜松2滴 + 迷迭香2滴

脚气，脚臭

洗脚水里面滴几滴没药精油

改善皮肤：涂抹，抗霉菌功能，抑制皮肤的炎症，对面疱湿疹豆疗效，滋润干裂皮肤，对皱纹有效。

乳液50毫升+没药4滴+乳香4滴+薰衣草2滴

咽喉肿痛

尤加利精油1滴 + 水300ml
使用方法：精油滴入水中摇晃均匀，漱口即可。不可吞服。

茶树2滴 + 安息香2滴
使用方法：40度左右的热水，滴入精油，蒸汽吸入

经常性打喷嚏，流感，呼吸系统，缓解头痛，集中注意力

尤加利精油2滴 + 薰衣草精油1滴 + 迷迭香精油1滴
使用方法：香薰

咳嗽

尤加利2滴 + 安息香3滴 + 热水1小盆
使用方法：40度左右的热水，加入精油，吸嗅蒸汽

丝柏1滴 + 薰衣草1滴 + 马郁兰1滴
使用方法：所有精油滴在面巾纸上，直接细嗅气味即可

呼吸道感染：

没药1滴+尤加利2滴+百里香2滴

乳香對肺非常好，是最適合治療呼吸道感染的精油之一。它是種有效的肺部殺菌劑，可以緩解咳嗽，治療支氣管黏膜炎（如慢性支氣管炎）等病症。

呼吸系统

蒸气吸入，对胸腔感染之支气管炎、肺部感染的喉咙痛、干咳也有效果。
檀香2滴+没药1滴+薰衣草2滴

对呼吸系统有益：感冒，咳嗽，鼻塞，头痛，偏头痛等
罗勒2滴+安息香1滴+松精油2滴

鼻塞，流鼻涕，呼吸不顺

松精油4滴+绿花白千层2滴+尤加利2滴
蒸汽吸入

减轻抑郁

佛手柑精油2滴 + 天竺葵精油1滴
细嗅：将精油滴在面巾纸上，深呼吸3口，在将面巾纸放在附近，以便随时闻到芳香

缓解焦躁

天竺葵2滴 + 薰衣草1滴 + 佛手柑3滴

壮阳催情

涂抹于生殖器官上，调节荷尔蒙分泌。也可直接用檀香精油点香薰灯达到效果。
檀香5滴 + 荷荷巴油5毫升 　

阳痿

依兰3滴 + 肉桂4滴
使用方法：熏香，每天最多熏3次，持续使用即可壮阳

茉莉精油3滴 + 姜精油6滴 + 芝麻油15ml
使用方法：调和均匀后，按摩背部等感觉舒服的地方

前列腺炎

杜松子精油1滴 + 芝麻油5ml + 尤加利（柠檬桉）精油2滴
使用方法：调匀后，涂抹于生殖泌尿器官处

丝柏精油3滴 + 薰衣草精油2滴 + 橄榄油5ml + 热水1盆
使用方法：调匀后倒入37-38度的热水，搅散后坐浴

专利配方：
一种缓解男性前列腺炎症的植物复方精油，取尤加利精油20－50％、茶树精油20－50％、熏衣草精油10－40％、百里香精油0.1－20％、迷迭香精油0.1－20％加工混合成原液；再以原液3－20％，基础油(橄榄油)80－97％融合制成植物复方精油。该种精油无毒副作用

杜松子2滴+尤加利2+茶树2滴+薰衣草2滴+百里香1+迷迭香1滴

1、 清洗：每日用茶树精油清洗生殖系统，每盆水里滴3-4滴，用温水，可坐浴。
2、 按摩：可用精油薰衣草、丝柏、尤加利、迷迭香、檀香、茶树。调配比例：熏衣草3滴+檀香3滴+尤加利2滴+迷迭香2滴+茶树2滴+基础油40滴。按摩下腹、背部下方和骶骨，前期量可以轻一滴，精油用量可减半，观察反应，再逐渐加量，以避免出现秘现反应。每日2-3次。

生殖系统

5滴檀香精油加入5毫升按摩底油中，涂抹于生殖器官上，可调节荷尔蒙分泌，其抗菌作用还可以净化及治疗生殖系统炎症，檀香对男性具有催情作用，并增加男性的自信和个人魅力。

膀胱炎

百里香精油2滴 + 迷迭香精油1滴 + 芝麻油5ml
使用方法：调匀后，涂抹于生殖泌尿器官处

龟头发炎

百里香精油2滴 + 绿花白千层精油1滴 + 橄榄油5ml
使用方法：调匀后，涂抹于生殖泌尿器官处

女性呵护：盆浴，改善阴道感染，治疗阴道炎。

用盛滿1000ml的溫水臉盆中，加入沒藥精油4-5滴
没药2滴+天竺葵3滴+洋甘菊3滴
没药2滴+薰衣草3滴+洋甘菊3滴

乳香對尿道和生殖道的影響力很強，古人經常利用乳香來治療這些部位的疾病。它具有調理子宮的作用，患者需要時用乳香精油進行沐浴，或在腹部進行溫和的芳香按摩。

痔瘡

前段吃瑪卡過量導致痔瘡嚴重發作。每天解大便大量出血非常痛苦。因為怕痛連續三天禁食。一直折騰了半個多月。用完了加拿大和澳洲的痔瘡膏也沒有解決。 ！幸虧給了絲栢 、天竺葵兩樣精油給我局部塗抹。僅3個鐘頭症狀明顯緩解。我感到發炎部位沒那麼痛了，水腫也在縮小。用了一天、也敢解大便了（當然還痛！）繼續用了一周完全恢復正常飲食了[微笑]
