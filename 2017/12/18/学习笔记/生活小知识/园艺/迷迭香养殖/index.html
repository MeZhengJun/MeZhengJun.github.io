迷迭香养殖

发表于 2017-12-18 更新于 2018-02-22 分类于 园艺 阅读次数： 本文字数： 1.5k 阅读时长 ≈ 1 分钟

养殖：

喜干旱怕湿热，10-25摄氏度，不得低于-2.8摄氏度。喜欢日照，夏季略微遮阴，富含沙质，排水良好的土壤。
少量浇水，宁干勿湿，一次浇透。定期通风。春秋两季适合插枝，把枝叶剪成约10厘米左右
土壤碱性越高，迷迭香就越香。如果土壤酸性过高，可将一些石灰埋进土里

刚买回来的迷迭香，在装盆之前，先要用一些东西垫在花盆底部，防止烂根，增加透气性和渗水性。比如：碎瓦片，陶粒，塑料泡沫等，要注意迷迭香喜欢碱性土壤。刚刚查了一下核桃壳ph值是4.89弱酸性，不适合给迷迭香垫底了。

剪取插枝的长度请参考下面表格:

季节 剪下的部分 长度
春季或初夏 绿色柔嫩的新枝 5～10cm
夏天到秋天 灰白部分的枝 10~15cm
秋天 棕色坚硬成熟的枝 15~ 35cm

如果用浸泡法的话用手指搓掉插枝下半截1/2的叶子，如果用插土法的话去掉枝径下1/3的叶子。如果插枝太娇嫩可用小夹子拔掉叶子。

插水法

  • 将处理好的插枝放入透明容器中，注水到将将达到叶子下面的高度。
  • 然后放到阳光充足的窗台上。头1个星期每2到3天换一次水，之后可待水变得浑浊或有明显沉淀物时再换。
  • 5到8周后插枝下半部会有白色的细根生出，继续浸泡 ，等根发到至少有5厘米长，有7，8条根就可以开始移栽了。但是待根越多移栽后的成活率就越高，生长速度会越快。

插土法

  • 把松软的细土填到花盆里，用手把盆栽土*压紧压平，但是不要太紧。
  • 浇透水。
  • 将处理好的插枝直接插入松软的细土里，间隔至少4厘米，放在阳光充足的地方。
  • 第一次浇水要浇透，往后每周浇透一次，平日保持土壤润泽。如果所在的地方气候干燥一定要在花盆里插上高于迷迭香插枝的木棍或筷子，然后覆盖上带孔的塑料袋，塑料袋不要接触插枝。
  • 大约6到7周后迷迭香会生根。生根后去掉塑料袋，小心用小铲子或叉子松动盆中的土壤，取出插枝，不要拔出插枝避免把根弄断了。
  • 移栽到单独的花盆里，浇透水。此时的插枝因为很久没有得到营养，而且还生了很多根消耗很大，浇水时可以适当补一点营养液。
  • 载到花盆里的插枝还要小心照看至少2周左右，定植后可根据所在地方的气候和条件决定继续室内或室外养植。

结论

两种方法比较过后还是发现水浸法成功率高得多，而且也简单得多，很容易观察生根的进展。最快的小插枝才插水了15天左右就开始生根了。插土法6根里只有2根有生根，所以本人还是建议用插水法。

10cm左右的半坚硬插枝通过浸泡法生根的成功率是最 高的。迷迭香的插枝种植最适合在春末夏初，花开败后进行。

迷迭香叶子干枯原因分析

  1. 缓苗期导致迷迭香叶子干枯
    刚买回来的植株都会经历一个缓苗期，就是适应环境，在这一时期可能出现叶子发黄、干枯的现象。过一段时间后，会逐渐缓过来。只是在此期间，将植株放在阴凉的环境中，减少浇水、施肥，等其适应环境后，正常养护就好。

  2. 通风不当导致迷迭香叶子干枯
    迷迭香适合生长在温暖的环境中，但环境闷热通风不良，会导致生长缓慢。迷迭香的生长速度本来就慢，如果温度较高，通风不当，生长更加受到抑制。叶子变黄、干枯只是开始，如果不及时通风，整个植株都会变成杂草。

  3. 光照不足导致迷迭香叶子干枯
    迷迭香喜欢阳光，每天至少要接受6小时左右的阳光照射，若光照不足，叶片光合作用产生的叶绿色较少，叶片暗黄，植株自身的养分也不充足，时间一长，叶片会逐渐干枯，并且容易感染各种病虫害。

  4. 病虫害导致迷迭香叶子干枯
    迷迭香的叶片窄而茂盛，很难观察到叶片上有虫子，其实有些虫子专门钻空子，附着在狭窄的叶片背面，很难观察到。所以在养护中，保持环境整洁，经常通风，定时喷洒杀菌药剂，及时将盆土表面的枯叶清理掉，剪掉过多和老化的枯枝叶。
