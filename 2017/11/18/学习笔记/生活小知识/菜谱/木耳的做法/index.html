木耳的做法

发表于 2017-11-18 更新于 2021-12-07 分类于 菜谱 阅读次数： 本文字数： 422 阅读时长 ≈ 1 分钟

做法和图片都是下厨房里摘录的，这里是为了方便自己查看。
主要是涉及木耳的做法，简单，好吃。

白菜炒木耳

用料

白菜，木耳，葱，姜，花椒，八角，老抽，白糖，醋，盐，鸡精

做法

  1. 白菜用手撒片
  2. 木耳用温水泡五分钟变软，摘去根部，洗净，葱、姜切丝
  3. 锅中放油，小火加热，放入花椒、八角炒出香味，放入葱、姜炒出香味儿，再放入白菜片，大火翻炒均匀
  4. 白菜片炒至微微变软时，倒入老抽翻炒均匀，放入白糖、醋适量
  5. 放入泡好的木耳，翻炒几分钟，撒少许盐、鸡精翻炒均匀，出锅即可

凉拌爽脆木耳

用料

黑木耳，大蒜，干辣椒，香菜，黄瓜，粉丝，橄榄油
花椒，生抽，香醋，芝麻油，糖，盐，熟白芝麻

  1. 木耳清水泡发，洗净后撕成小朵
  2. 大蒜切末，辣椒和香菜切成小段。
  3. 取一小锅加适量水烧开，入黑木耳焯烫1分钟左右捞出，过凉水洗去表面粘液，沥干后待用。
  4. 炒锅内倒少许橄榄油加热，放花椒爆出香味后滤出花椒，在将葱姜蒜放入爆香，倒入生抽，香醋，盐和糖调匀成味汁。
  5. 将调好的味汁倒入木耳，再淋上芝麻油，加适量香菜和白芝麻拌匀即可。
