gradle

Spring Boot 2.2 增加了对Java 13的支持 对于老版本的Java 11、Java 8依然支持良好。所以，对于还在使用Java 8的用户也可以安心的升级使用。

配置 VPN 代理

  • 配置 VPN
    我这里使用的是 shadowsocks。socks5 配置为 127.0.0.1:1086

  • 配置 IDEA
