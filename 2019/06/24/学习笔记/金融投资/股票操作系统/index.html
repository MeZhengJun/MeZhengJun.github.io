股票操作系统

Posted on 2019-06-24 | In 读书

经验的总结。

操作原则

  1. 确定大盘的趋势，根据所处的趋势采取相应的策略，个股和大盘共振才能使赢的概率变大。
  2. 确定入场的性质，支撑点反弹（短期操作，不可贪心），趋势形成（阶段性操作，根据卖出信号操作）
  3. 判断错误，及时认错，注意风险回报比是否合算。
  4. 卖出信号：
    • 趋势中，出现短期均线纠缠，说明原有趋势出现了不确定性。

记录每笔交易
月底分析每笔交易，总结经验，不断的学习改进。

止损的方法：

  1. 百分比止损，8%以内
  2. 价格模式止损：根据价格形态判断趋势转变
  3. 平均真实振幅止损
  4. 抛物线止损，反转指标止损

大盘趋势的判断

大盘趋势分为：强势上涨，弱势上涨，强势下跌，弱势下跌，整理
在不同的趋势中关注点不同。

  1. 强势市场中关注突破
  2. 弱势市场中关注趋势的持续性
  3. 整理市场中关注反转迹象

强势上涨的判断：

  1. 20，50 天均线金叉
  2. 20，50 均线都在上涨
  3. 20，50 均线间距较大或正在变大
  4. 价格回调只会回调到 20 天均线，最多回调到 20，50 均线之间，不会回调到 50 均线之下。

如何操作

寻找长期整理并刚刚突破创新高的股票。这时要注意指标的顶背离，避免假突破。

弱势上涨的判断

  1. 20 均线大多数时间是在 50 均线上方
  2. 50 均线上涨，但 20 均线上下跳动（大部分时间是上涨的）
  3. 20，50 均线间距经常变化
  4. 回调会跌到 50 均线（有时会跌到 50 均线以下，通常时间很短）

如何操作

趋势很强（通常比整体市场强），但会回调到支撑区，使用超卖指标确认回调，如果这时看到反转 K 线，就可以考虑入场。

强势下跌趋势的判断

最少在三个月内最高点越来越低。

  1. 20 均线在 50 均线下方
  2. 20，50 均线都在下跌
  3. 20，50 均线间距较大或正在变大
  4. 反弹只会上涨到 20 均线，最多反弹到 20，50 均线之间

如何操作

寻找长期整理并刚刚突破创新低的股票。这时要注意指标的底背离，避免假突破。
同时要注意底部的大量买入，可能会有 V 型反转。

弱势下跌趋势的判断

  1. 20 均线大多数时间是在 50 均线下方
  2. 50 均线下跌，但 20 均线上下跳动（大部分时间是下跌的）
  3. 20，50 均线间距经常变化
  4. 反弹会涨到 50 均线（有时会涨到 50 均线以上，通常时间很短）

如何操作

弱势下跌是一个很好的趋势交易市场，寻找下跌强度比较强，但反弹到阻力区的股票。通过超买指标确认，日线图等待反转 K 线入场。

震荡市场的判断

震荡市场即可以做多，也可以多空，尤其是趋势线向上、下突破的时候。
震荡市场价格在阻力位和支撑位之间来回跳动，可以使用 kdj，cci，rsi 来确定超买，超卖。

通常震荡市场不太好判断，下面是一些参考原则

  1. 20 均线在 50 均线上下的时间差不多
  2. 50 均线几乎是水平的，20 均线则有涨有跌
  3. 20，50 均线间距变化很大
  4. 两条均线之间会发生很多上涨和下跌现象

如何操作

震荡市场第一次出现时结束了原有的的趋势市场。这时不易判断。
震荡趋势形成后，可以使用趋势线界定交易区间，使用超卖，超买，背离指标，当价格到了支撑，阻力点附近时可以做反转，注意反转需要趋势线确认。
指标背离是很好的确认波动顶部和底部的方法。

做多的操作模式

上涨趋势的股票，至少在 3 个月内底部是越来越高的。

回调

用在强势，弱势上涨的市场

关键买入信号

  1. 回调到了 20 均线或 20，50 均线之间，或 50 均线，如果正好跌到了上涨趋势线就更好了。
  2. 价格回调时，kdj 必须快速下跌到超卖线 20 以下，确认超卖状态。
  3. K 线必须有反转形态（倒锤形，阳包阴，十字星等），否则就在等等，第二天继续观察

辅助条件，增加概率：

  1. 回调到支撑点的成交量比均值小
  2. 回调时收盘价不会低于 50 日线
  3. 支撑上涨的趋势线没有被刺破
  4. 任何指标没有顶背离
  5. 日线没有双顶形态，周线价格不会低于 50，200 周均线


上升楔形，三角形，旗形

用在强势，弱势上涨，震荡市场

  1. 弱势上涨，近期（20 个交易日内）创新高（至少 3 个月的）
  2. 形态末端要越来越窄，形态的交易区间不能是上涨的，可以震荡，或略微下跌。
  3. 形态要持续 7-20 个交易日这个很重要，并且不会延伸到 50 日线或以下。
  4. 如果突破上趋势线就是买入信号，低于下趋势线就是败了。去掉

辅助条件，增加概率：

  1. 形态内成交量小于均值
  2. 形态内 K 线不少于 7 根，也不多于 15 根
  3. 形态内收盘价不低于 50 日线
  4. 形态内阳线比阴线多
  5. 形态内 RSI，CCI，OBV 至少 2 个指标是上涨的
  6. 周线价格不会低于 50，200 周均线


底背离

用在振荡，弱势上涨，弱势下跌市场

  1. 确认股票在长期上涨趋势中
  2. 价格在 50 日线以下；价格回调时最少出现了 2 个低点，间隔至少 5 个交易日；伴随着 2 个或更多指标的底背离(MACD，MACD 柱，KDJ，RSI，OBV，CCI)
  3. 如果得到 200 日货趋势线或之前价格的支撑就更好了。
  4. 满足前面几点的情况下，日线图上 K 线出现了上涨形态（十字星，锤形，阳包阴，等等）出现买入信号

辅助条件，增加概率：

  1. 最后的最低价的成交量大于均值
  2. 最后的最低价在价格支撑区，长期趋势线支撑区或主要均线支撑区附近
  3. 至少两个指标底背离
  4. 最低点离 20 日线至少有 20%
  5. 周线价格不会低于 50，200 周均线


杯型突破

  1. 股票至少创了 3 个月的新高，必须在前一个整理新高（至少 5 个交易日的整理）之后的 20 个交易日再创新高。必须在最近的短期的波动之后产生的新高
  2. 最新的收盘价必须是明显的最高点，至少 3 个月内
  3. 最新的收盘价不能离 52 周最低点太远。收盘价/52 周最低价 <= 3.0
  4. 突破时 OBV 至少是 3 个月以来的最大值。
  5. 突破创新高那天的 K 线必须是阳线

辅助条件，增加概率：

  1. 突破当天成交量大于均值
  2. 突破的最高价是 6 个月或以上的最高价
  3. 突破后会有一段时间的整理，有几次向最高价的假突破


底部上涨突破

用在弱势上涨，强势，弱势下跌，振荡市场

  1. 股票必须在明显的强势或弱势下跌趋势中
  2. 至少在 30 日内价格有某种整理形态：三角形，矩形
  3. 要回避上升楔形，这个是中继整理形态，会继续下跌
  4. MACD 必须出现一系列比较高的低点，价格方面不是必须的
  5. OBV 必须上涨到 OBV 上趋势线以上

辅助条件，增加概率：

  1. 突破当天成交量大于均值
  2. 价格还没有突破看涨的底部
  3. 底部持续的时间大于最低的要求时间
  4. 阳线比阴线多
  5. 周线上底部在主要的周均线 20，50，200 附近





做空的操作模式

下跌反弹

用于强势，弱势下跌市场

  1. 反弹到 20，50 均线附近，如果均线和下降趋势线一起就更好了
  2. KDJ 必须快速上涨或超过 80，称为超买状态
  3. 反弹到均线时必须出现下跌的 K 线形态（十字星，阳包阴，墓碑，上吊线等），如果 K 线是阳线就要继续等待。

辅助条件，增加概率：

  1. 反弹到阻力的成交量低于均值
  2. 回调时的收盘价不高于 50 天均线
  3. 下跌趋势的趋势线没有被刺破
  4. 指标没有出现底背离
  5. 日线没有形成双底，头肩底等反转形态
  6. 周线价格不会低于 50，200 周均线


顶背离

用在振荡，弱势上涨，弱势下跌市场

  1. 股票趋势是长期下跌趋势
  2. 价格在 50 天均线之上；价格有两个高点且间距 5 个交易日以上
  3. 指标顶背离（MACD，MACD 柱，KDJ，OBV，CCI，RSI）
  4. 最新的最高价要和 200 均线，趋势线的阻力区或以前的价格阻力一致，越接近越好
  5. 日线上出现下跌 K 线（十字星，阳包阴，墓碑，上吊线等）触发做空信号

辅助条件，增加概率：

  1. 最新的最高价处成交量大于均值
  2. 最新的最高价在之前的价格阻力区附近
  3. 指标的背离比较强，至少两个指标出现背离
  4. 最新的最高价离 20 日线至少有 10%的距离
  5. 周线价格不会低于 50，200 周均线


向下跳空

用于强势，弱势上涨，振荡市场

  1. 50 日均线稳步上涨，没有大的回调；价格至少有 40 个交易日在 50 日均线之上
  2. 向下跳空缺口没有被补上
  3. 价格低于跳空缺口的 K 线，触发信号

辅助条件，增加概率：

  1. 跳空当天成交量大于均值
  2. 跳空以前的最高价在之前的价格阻力区
  3. 跳空前出现了顶背离


杯型突破

用在强势下跌市场

  1. 股票跌到了三个月的最低价，20 个交易日内必须出现第二的低点（收盘价）。
  2. 最新的收盘价必须是明显的最低点，至少 3 个月内的
  3. 新的最低收盘价不能离股票 52 周最高点太远，52 周最高点/新的收盘价 <= 2.0
  4. OBV 出现至少 3 个月内的最低值
  5. 向下突破新的最低价的 K 线必须是阴线。

辅助条件，增加概率：

  1. 跳空当天成交量大于均值
  2. 向下突破后的最低价是 6 个月以来的最低价
  3. 向下突破后的一段时间盘整，会有几次向最低点的假突破


上升楔形向下突破

用于强势，弱势上涨，振荡市场

  1. 必须处于某种明显的上涨趋势中
  2. 上涨趋势必须在越来越窄的区间上下波动
  3. MACD 必须顶背离，OBV 跌破低点连接的趋势线，K 线为阴线

辅助条件，增加概率：

  1. 出现信号当天成交量大于均值
  2. 价格还没有突破楔形
  3. 上升楔形比上涨三角形要好
  4. 阴线比阳线多
  5. 在形态内 RSI，CCI，OBV 至少有两个指标是下跌的
  6. 周线图上，形态在主要的周均线附近，20,50,200 日均线
