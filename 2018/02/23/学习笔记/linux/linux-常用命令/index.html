所有的信息都输出到同一个文件中：

make xxx > build_output_all.txt 2>&1
其中的 2>&1 表示错误信息输出到 &1 中，而 &1，指的是前面的那个文件：build_output_all.txt 。
注意：上面所有的1,2等数字，后面紧跟着大于号’>’ ，中间不能有空格
