第一章

识记

企业：一般来说是指从事商品的生产，流通或服务等活动，为满足社会需要进行自主经营，自负盈亏，承担风险，独立核算，具有法人资格的基本经济单位。
特征：1 以价值增值作为其经济活动的目的 2 从事商品生产和流通的经济组织 3 自主经营，独立核算，自负盈亏
个体企业特征：由业主个人出资兴办，业主自己直接经营的企业。业主享有企业的全部经营所得，同时对企业的债务负有完全责任。
合伙企业特征：由两个或两个以上的个人共同出资，通过签订协议而联合经营的企业。合伙人分享企业所得，对营业亏损共同承担无限连带责任。（不是法人）
公司制企业特征：1 企业是法人，2实行有限责任制度，3所有权和经营权分离
按企业所属的经济部门划分：农业，工业，金融，商业企业等
按生产力要素比重划分：劳动密集型，技术密集型，知识密集型，资源密集型企业等
按企业的经营规模划分：大型，中型，小型企业
按企业的财产构成和所负法律责任划分：个体企业，合伙企业，公司制企业
管理：管理者或管理机构，在一定范围内，通过计划，组织，领导，控制等工作，对组织的资源进行合理配置和有效整合，以实现组织预定目标的动态过程。
管理的职能：计划，组织，指挥，协调和控制组成
计划职能的含义：是管理的首要职能，对既定的目标进行具体安排，作为全体员工在一定时期内的行动纲领，并规定实现目标的途径，方法的管理活动。
组织职能的含义：为了有效地完成既定的计划，通过建立组织机构，确定职能，职责和职权，协调相互关系，合理配备和使用企业资源的管理活动。
组织工作的基本原则：1有效性，2统一指挥，3责权利相一致，4集权与分权相结合，5弹性，6协调原则
领导：是领导者影响并感召个人和群体去追求某些目标的行为和过程。
控制职能的概念及方式：是指接受组织内外的有关信息，按既定的目标和标准对组织进行监督，检查，发现偏差，采取正确的措施使组织按照既定的计划进行，或适当地调整计划，以达到预期目标的管理活动。
控制的方式：1预先控制，2现场控制，3反馈控制
企业管理的概念：是企业管理者或管理机构，为实现企业价值最大化的目标，通过计划，组织，领导，控制等活动，对企业的生产经营活动及其资源进行合理配置和有效整合的动态过程
古典管理理论的代表人物及理论名称：

“科学管理之父”的泰罗的科学管理理论
“管理过程理论之父”法约尔的管理过程理论《工业管理和一般管理》
“组织管理理论之父”韦伯的管理组织理论《社会和经济理论》

行为科学理论的代表性理论成果：马斯洛的需求层次理论，赫茨伯格的双因素理论，麦克莱兰的激励需求理论，麦格雷戈的X理论-Y理论。
“管理理论丛林”的产生及有影响的代表学派。
二战后的40-60年代，美国的经济水平得到了大幅度的发展。巴纳德的社会系统学派，西蒙的决策学派，德鲁克的经验（案例）学派等。到80年代初发展为11大不同学派，孔茨称其为“管理理论丛林”
战略管理理论的产生
网络企业的特点：
1 所占的现实空间非常有限。
2 计算机化和网络化的企业
3 全天候运作的企业
4 信息技术和信息产品应用型的企业
5 高知识型的松散企业
企业的网络化管理的含义：1 目的是构建一个客户，供应商，企业员工共享的信息平台，提高管理水平，从而加快企业对客户需求的反应速度。2 包括对企业支撑网络的管理，网络人才的管理，网络的安全防范管理等内容。3 需要实施企业管理的根本变革
准时制（JIT)：基本思想是“只在需要的时候，按需要的量，生产所需的产品”，也就是追求一种无库存，或库存达到最小的生产系统。
制造资源计划（MRPII）：出现于20世纪70年代末期的，以企业资源优化配置，确保企业连续，均衡地生产，实现信息流，物流与资金流的有机集成和提高企业整体水平为目标，已计划与控制为主线，面向企业产，供，销，财的现代企业管理思想和方法。
并行工程（CE）：集成地，并行地设计产品及其相关的各种过程（包括制造过程和支持过程）的系统方法。这种方法要求产品开发人员从设计一开始就考虑产品整个生命周期中从概念形成到产品报废处理的所有因素，包括质量，成本，进度，计划和用户的要求。
计算机集成制造系统（CIMS）：可以把计算机在各项专业管理中的运用结合起来，改变计算机在管理和运用中相互分割，各自为政的局面。CIMS的中心概念是“集成”，他把一些成熟的管理办法和技术转化为数学模型和软件包，形成四大方面的系统，即管理信息系统，设计与开发信息系统，生产自动化信息系统和质量控制系统，CIMS对这四个系统实现了“集成”和一体化，从而大大提高了工作效率。
企业资源计划（ERP）：由美国加特纳公司在90年代初提出的，以客户需求为导向，以实行企业内外资源优化配置，消除生产经营过程中一切无效的劳动和资源，实现信息流，物流，资金流，价值流和业务流的有机集成和提高客户满意度为目标，以计划与控制为主线，以网络和信息技术为平台，集客户，市场，销售，采购，计划，生产，财务，质量，服务，信息集成和业务流程重组等功能为一体，面向供应链管理的现代企业管理思想，方法和工具。
分销资源计划（DRP）：是管理企业的分销网络的系统，目的是使企业具有对定单和供货具有快速反应和持续补充库存的能力。通过网络系统，为企业的业务经营及贸易伙伴的合作提供了一种全新的模式。供销商之间可以实时地提交订单，查询产品供应和库存状态，并获得市场，销售信息及客户支持，实现端到端的供应链管理，有效地缩短供销链，使企业的工作效率和业务范围得到有效的提高。
企业内部网和互联网（Intranet&Internet）：互联网是覆盖全球的计算机互联网，是信息高速公路的主体。强调网络间的各种连接。Intranet是互联网的进一步发展，是建立在互联网上的企业内部网，它可以通过接入的方式成为互联网的一部分，也可以独立自成体系。最大的好处是企业内部的信息交流。强调企业内部门间的连接
电子数据交换系统（EDI）：是一种以电子方式交换数据的技术，即将业务文件按一个公认的标准从一台计算机传输到另一台计算机的电子传输方法。

领会

现代企业的产生和发展的三个时期。
企业的发展过程：

  1. 手工业生产时期16-17世纪，从封建社会的家庭手工业到资本主义初期的工场手工业时期
  2. 工厂生产时期18世纪50-60年代，
  3. 现代企业生产时期19世纪末，20世纪初

从工厂生产时期过渡到现代企业生产时期的主要特征。

  1. 生产规模空前扩大，产生了垄断组织,如：托拉斯，辛迪加，康采恩
  2. 不断采用新技术，新设备，不断进行技术革新，使生产技术有了迅速发展
  3. 劳动分工越来越精细，协作关系复杂，严密
  4. 建立了一系列科学管理制度，并产生了一系列的科学管理理论
  5. 所有者与经营者分离，企业里形成了一支专门的工程技术队伍和管理队伍
  6. 企业之间的竞争激烈，加速了企业之间的兼并与收购，使生产进一步走向集中
  7. 企业的社会责任改变，不仅在整个社会经济生活中的作用越来越大，同时渗透到政治，军事，外交，文化等各个方面

企业管理的目标：实现企业价值的最大化
计划的种类：1长期，中期，短期计划，2战略计划和具体行动计划，3综合计划，部门计划和具体项目计划，4指令性计划和指导性计划
计划的内容和程序：

  1. 分析和预测外部环境，内部条件和未来的变化趋势
  2. 制定中长期和近期目标
  3. 评价和选择可行性方案，进行决策
  4. 编制综合计划和各项专业计划（如生产计划，销售计划等）
  5. 检查计划的执行情况

组织职能的内容：

  1. 将为实现管理目标所必须进行的各项活动，根据其内在的联系及工作量进行分类组合，设计出各种基本职务和组织机构
  2. 规定各种职务和各个组织结构的责，权，利及其上下左右的关系，并且对组织结构，责任制度，职责条例，工作守则等形式加以说明
  3. 选拔和调配适合的人员担任相应的职务，并授予执行职务所必须的权利，是每个人都能充分发挥作用
  4. 通过职权关系和信息系统把各个组织机构联成一个严密而又有活力的整体
  5. 对管理组织系统内的职工进行教育培训和智力开发，使他们的知识不断更新，更有效地完成自己所承担的工作

领导工作的内容：1权利的形成和运用，2指导，3激励，4沟通，5协调，6营造组织气氛，建设组织文化
控制职能的必要性：1组织环境的不确定性，2组织活动的复杂性，3管理失误的不可避免性
企业再造理论和实践的兴起
全球化和网络时代的企业学习型组织，长寿公司及虚拟组织

网络时代企业环境变革的特点：

  1. 信息技术特别是网络技术日臻完善
  2. 经济的全球化
  3. 顾客的需求更加多样化和个性化
  4. 科学技术的发展日新月异
  5. 变革的速度越来越快
  6. 知识成为企业最重要的资本

网络时代企业面对的机遇：

  1. 企业可以更好地满足顾客的个性化需求
  2. 企业可以降低交易成本
  3. 企业可以减少库存
  4. 企业可以使合作竞争战略更便利地实施
  5. 提高获取知识，应用知识的能力

  4. 企业关键的成功因素将会改变
  5. 企业资源配置的方式和界限将会发生重大变化

网络时代企业管理范围的拓展：

  1. 网络企业管理
  2. 企业的网络化管理
  3. 更加重视以知识资本为核心的无形资本的管理
  4. 企业管理的范围拓展到整个供应链

企业管理内容的创新：

  1. 企业战略管理的创新
  2. 企业组织管理的创新
  3. 网络营销成为营销管理的重要内容
  4. 敏捷制造成为企业生产运作管理最重要的方式
  5. 企业财务管理将向战略型，集成化方向发展
  6. 回归人本管理，重视人力资源管理
  7. 知识管理成为企业管理的重要内容
  8. 更加重视文化管理

企业管理方法和手段的创新：

  1. 准时制（JIT）Just in Time
  2. 制造资源计划（MRPII）Manufacturing Resources Planning
  3. 并行工程（CE）Concurrent Engineering
  4. 计算机集成制造系统(CIMS) Computer Integrated Manufacturing System
  5. 企业资源计划(ERP) Enterprise Resources Planning
  6. 分销资源计划(DRP)Distribution Resource Planning
  7. 企业内部网和互联网(Intranet Internet)
  8. 电子数据交换系统(EDI) Electronic Data Interchange

第二章

识记

战略的性质：书上无定义，描述的只是战略的重要性
企业战略：是着眼于企业的未来，根据企业的外部环境和内部资源的条件，为求得企业的生存和发展而进行的一系列带有全局性，根本性和长远行的谋划。
分为三个重要层次：企业总体战略，事业层战略，职能战略

企业战略管理：依据企业内外环境变化制定战略，实施战略，并根据结果的评价和反馈来调整，制定新战略的过程。
企业战略管理特点：1. 整体性；2. 长期性；3.权威性；4.环境适应性；
公司环境的基本概念：任何一个企业和社会组织都是存在于环境之中的。企业环境可分成微观环境和宏观环境。
公司宏观环境的构成：包括那些影响企业微观环境中所有行动者的较广泛的社会力量或因素，包括人口的，经济的，技术的，政治的，法律的以及社会文化方面的力量和因素。

企业核心能力概念及特性：是企业独有的，能够支撑企业长期竞争优势的能力。它能为企业进入各类市场提供潜在机会，能借助最终产品为所认定的顾客的利益做出重大贡献，而且不易为竞争者所模仿。
特性：1 有价值的创造能力。2 异质性。3难模仿性。4不可交易性。5扩展性。6动态性。

（产业中的竞争结构）是决定产业竞争规划和激烈程度的根本因素。
产业中的基本竞争力量：顾客，现实竞争者，潜在竞争者，替代品压力，供应商

企业核心能力的来源：有两个互相补充的来源，1 企业所具有的资源，2企业的内聚能力。

市场细分：就是企业按照某个（些）“细分变数”把整个市场细分为若干需要不同的产品和市场营销组合的亚市场，其中任何一个亚市场都是由一个有着相似的欲望或者需要的顾客群所组成，都可被企业选作目标市场，这个目标市场，也就体现了企业战略的基本意图。
战略分析的主要内容和方法包括：市场细分，产业分析，竞争对手分析，竞争优势分析。
企业价值活动可以分为两大类：基本活动和辅助活动。
基本活动包括内部后勤，生产经营，外部后勤，市场销售，服务；
辅助活动包括人力资源管理，技术，开发，采购

经营战略的类型包括：发展型企业战略，稳定型战略，紧缩型战略，组合战略。

竞争对手分析法：

  1. 竞争对手的各期目标和战略。
  2. 经营状况和财务状况分析。
  3. 技术经济实力的分析。
  4. 领导者和管理者背景分析。

竞争优势分析法：决定一个企业持续竞争优势的是企业实际开展的设计，生产，营销，交货，人力资源管理等相互分离但又相互联系的活动或工作。
资源分配：在确定资源组合方面可以使用波士顿矩阵作为分析工具，还可以对现行的组合进行分析
根据业务增长率和竞争地位将企业的各战略经营单位分为四类：明星类，金牛类，问号类，狗类

战略规划的任务：将战略分析和战略选择的结果进一步体现在产品组合和资源分配上。
价值属性的概念：是指用户在选择，购买，使用企业产品的过程中所感受到的满足的种类以及各类满足的程度。显然，价值属性必须以产品属性为基础或依据，价值属性是产品属性在顾客心目中所形成的心里映射或者说主观感受。

迈克尔-波特教授在其名著《竞争战略》中指出竞争战略的基本类型：

###识记
组织的基本含义：企业组织是由为了实现共同的目标的众人建立的具有规范的秩序，职权层级，沟通系统和成员协调系统的一种动态复杂系统，它在与环境发生相互作用的过程中，极力维持自身及环境间的动态平衡。

从职能或活动角度来看：组织就是通过设计和维持组织内部的结构和相互之间的关系，使人们为实现组织的目标而有效地协调工作的过程。
企业组织的性质：

  1. 是根据一定的目的建立的，组织是实现组织目标的有效方式。
  2. 组织需要管理。
  3. 在组织中要确立一种规范的秩序，即制度。
  4. 组织是一种权责结构。
  5. 在组织的不同权责结构之间存在着沟通和协调。
  6. 组织的基础是活动，组织是为了完成某种目标而将有关人，物等组织起来。
  7. 组织是一个动态的开放系统。

企业组织管理的概念：为了有效地配置企业内部的有限资源，为了实现一定的共同目标而按照一定的规则和程序构成的一种责权结构安排和人事安排，其目的是在于确保以最高的效率，实现组织目标。
组织管理的基本内容：就是设计，建立并保持一种组织结构，包括：

  1. 组织设计
  2. 组织运作
  3. 组织调整

直线制结构的概念：是最早最简单的一种组织结构形式。组织中的每一个管理层级都具有直线职权，能够管理下属人员，同时又受上级管理人员指挥。
职能制结构的概念：在组织中设置若干职能专门化的机构，这些职能机构在自己的职责范围内，都有权向下发布命令和指示。

直线职能制结构的概念：综合直线制和职能制两种类型组织的特点而形成的组织结构形式。与直线制的区别在于，职能机构只是作为直线管理者的管理和助手，它们不具有对下面直接进行指挥的权力。

事业部制结构的概念：组织按地区或所经营的各种产品和事业来划分部门，各事业部独立核算，自负盈亏，适应性和稳定性强，有利于组织的最高管理者摆脱日常事务而专心致力于组织的战略决策和长期规划，有利于调动各事业部的积极性和主动性，并且有利于企业对各事业部的绩效进行考评。

矩阵型结构的概念：这是一种把按职能划分的部门按产品，服务或工程项目划分的部门结合起来的组织形式。在这种组织形式中，每个成员既要接受垂直部门的领导，又要在执行某项任务时，接受项目负责人的指挥。

控股型结构的概念：是一家企业对其他企业不进行直接的管理和控制，而是代之以持股控制。被持股单位仍保留具体业务的自主经营权和独立的法人地位。这些企业之间形成了母企业，子企业和关联企业的关系。

网络时代，企业组织变革的特点：组织结构扁平化，柔性化，虚拟化，网络化
企业组织结构扁平化的概念：指减少管理层次而扩大企业管理幅度，组织结构形态由标准的金字塔型向圆筒型转化。
组织结构柔性化的概念：是指组织结构上不设置固定的和正式的组织机构，而代之以一些临时性的，以任务为导向的团队式组织。
企业组织虚拟化的概念：包括两方面含义：1企业组织内部的虚拟化，2企业组织之间的虚拟化。内部虚拟化是指将企业组织传统的运作方式改为以IT网络为基础来运作，从而实现企业组织的虚拟化。组织间的关系虚拟化是指具有不同功能的多个企业组织为了实现共同的目的，围绕一个核心组织，以信任和契约为基础，通过IT网络联结起来，形成一种临时合作的组织。
企业组织网络化的概念：是指用一种新的组织模式代替实行传统组织模式。
网络型组织的概念：由多个独立的个人，部门和企业为了共同的任务而组成的联合体，它的运行不靠传统的层级控制，而是在定义成员角色和各自任务的基础上通过密集的多边联系，互利和交互式的合作来完成共同追求的目标。

企业再造的概念：为了在衡量绩效的关键指标上取得显著改善，从根本上思考，彻底改造业务流程。其中，衡量绩效的关键指标包括产品和服务质量，
顾客满意度，成本，员工工作效率等。
含义：

  1. 需要从根本上重新思考业已形成的基本信念
  2. 以顾客为中心是出发点
  3. 是一次彻底的变革
  4. 通过业务流程再造可望取得显著的进步
  5. 从重新设计业务流程着手
  6. 实质上是管理和组织的创新
  7. 实施人本管理
  8. 重视信息技术的应用

三C理论：顾客(Customer)，竞争(Competition)，变化(Change)

共同愿景包括：共同目标，价值观和使命三个要素

市场营销的概念：在变化的市场环境中，为满足消费需要，实现企业目标而进行的商务活动过程，包括市场调研，选择目标市场，产品开发，产品定价，渠道选择，产品促销，产品存储和运输，产品销售，提供服务等一系列与市场有关的企业业务经营活动。
