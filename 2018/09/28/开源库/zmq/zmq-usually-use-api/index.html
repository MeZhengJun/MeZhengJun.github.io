zmq_proxy

在当前线程内开始ZMQ内置代理。
代理将一个前端socket连接到一个后端。一般来说，数据会从前端流向后端。根据socket的类型，数据可能向相反的方向流动。方向指示概念上的；代理是完全对称的，在技术上前端和后端没有什么区别。
zmq_proxy()函数在当前的线程空间中运行，并且只有在当前使用的context被关闭之后才会返回。
