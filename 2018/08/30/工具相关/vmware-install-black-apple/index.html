停止 VMware 服务

VMWare 安装完成后，打开任务管理器：win+R，输入 services.msc
将关于 VMware 的服务全部停止运行

进一步进行系统安装

按操作继续安装，多会遇到下面的提示 ＯS X Base System”上没有足够的可用空间进行安装 此时，点击屏幕上方的“实用工具 ->磁盘工具”。选中 “ vmware workstation SATA hard drive media ”,点击 “抹掉” 自定义分区名称，同时注意“格式”和“方案”的选择 退出 “磁盘工具”，再次进行安装，此时选择出现的分区。 点击“继续”，出现以下画面，如果空间足够，一般就没什么阻碍了…
