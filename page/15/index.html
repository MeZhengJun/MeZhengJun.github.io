MySql++ 库连接

MySql++

编译 MySql++，注意这里只能动态连接。静态链接官网说可能会有法律后果，具体没细看。

需要安装 MySQL Connector C 6.1，MySql++是基于这个接口库开发的。
C:\Program Files\MySQL\MySQL Connector C 6.1\include
路径下面有 vc 工程。打开设置 MySQL Connector C 6.1 的 include 和 lib，编译即可
mysql++-3.2.4\vc2008\
