.Net Framework 4.7.2 离线包安装时出现“无法建立到信任根颁发机构的证书链” 问题

证书链接

  • Win+R 打开运行，输入MMC，确定。
  • 文件→添加/删除管理单元
  • 添加证书

  • 选择 计算机账户→下一步→完成→确定

  • 展开证书→右键 所有任务→导入

导入成功后，再重新安装 .Net Framework 4.7.2 即可安装成功
