Github pull request

Posted on 2017-09-26 | In 工具相关

搭建这个静态博客也学了一些新的东西，挺好的。
Atom里面的图片插件markdown-image-assistant强制必须要输入一个图片路径附加在文件名后面，导致
Hexo自己生成的文件名文件夹不能使用。于是研究了一下，顺便提交了一个合并请求，这里记录一下吧。

  1. fork对方的项目
  2. clone到本地做修改调试
  3. 发起pull request给原来的仓库，让原作者看到你的修改
  4. 原作者review后，合并代码。至此整个流程就完成了。
