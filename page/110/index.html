精油疗法-基础油

发表于 2017-12-20 更新于 2017-12-25 分类于 健康 本文字数： 783 阅读时长 ≈ 1 分钟

因为纯精油的刺激性十分强烈，直接搽在皮肤上会造成伤害，因此单方精油无法直接抹在皮肤上（薰衣草和茶树精油可以用棉签小面积点涂），必须在基础油中稀释之后，才可以广泛用在人体肌肤上 。基础油的目的就是缓解，稀释，协助精油的使用。

基础油的来源

从植物的叶子，花朵，根茎或果实中萃取的非挥发性油脂，可润滑肌肤，能直接用于肌肤按摩，很多基础油本身就具有医疗的效果。
常用的三大基础油，荷荷巴油，甜杏仁油，葡萄籽油

基础油的选择需要是纯植物性油，对皮肤能够提供更好的滋润和养分。

此外要注意油的性状，分子大小与个人肤质，使用部位做搭配。

维生素/矿物质，必须脂肪酸含量高，分子小的油好些。
坚果，种子的植物油本身就具有特殊的易吸收、富含维生素的特性。

基础油 油质特性 颜色 适用部位 禁忌
荷荷巴油 油质较浓稠，不易氧化，液态植物油 金黄色 全身，头皮 无
甜杏仁油 油质轻，分子较葡萄籽油，荷荷巴油粗 黄色 任何肌肤 易长痘痘的油性肌肤避免使用
葡萄籽油 油质轻，分子小 偏绿色 脸部，易出油的部位 干性肌肤避免使用

按摩手法

顺着淋巴管的方向，颈部按摩由下向耳部按摩，上半身向腋下按摩，下半身由脚底向心脏方向进行。

相对好的使用方法是这样的：将适量的精华倒在手心，顺着面部淋巴排毒的方向，由鼻子中轴线向两边推，将整个面部分成额头、面颊、下巴三个区域整体涂抹。之后再用手掌顺着以上方向轻轻拉开精华液即可。如果你有时间的话，还可以加上适当按摩，增加吸收、促进循环。

保存

  • 避免直接光照和高温
  • 部分精油的特性类似于葡萄酒，即存储的年份越长，油的质量越佳。包括檀香木精油，玫瑰精油，茉莉花精油。
  • 未开封的基础油可以保存2年以上，开封后，进入空气产生氧化作用，容易变质，延缓的手段是加入维生素E胶囊。
  • 基础油除了可能有轻微坚果味外，几乎没有什么味道。到期的基础油含有一股明显的气味，可以轻易地闻到。
  • 通常可以将不用的基础油放于冰箱内密封存储，减慢氧化过程。

精油疗法-薰衣草精油

发表于 2017-12-20 更新于 2018-02-20 分类于 健康 本文字数： 1.1k 阅读时长 ≈ 1 分钟

薰衣草精油由薰衣草提炼而成的。可以清热解毒，清洁皮肤，控制油分，祛斑美白，祛皱嫩肤，去除眼袋黑眼圈，促进受损组织再生恢复。

要直接接觸皮膚的像是處理蚊蟲叮咬、小傷口，或是調入保養品還有睡前放鬆的薰香就用真正薰衣草

精油疗法-调和精油的方法

发表于 2017-12-20 更新于 2018-02-14 分类于 健康 本文字数： 496 阅读时长 ≈ 1 分钟

一般来说，身体按摩通常需要30ml的基础油，可以做一次全身，或局部多次按摩。

精油比例：

  • 面部保养2.5%左右（敏感皮肤1%）
  • 身体按摩3%-5%之间，持续使用以3个月为限
  • 一些特定目的，如：肌肉酸痛，感冒之类的涂抹，就可能达到10%以上（这个有待考证），不要持续使用超过2周

控制在5%之内比较安全，高了需要明确知道自己在做什么，否则可能会产生意想不到的后果。

青春痘-精油疗法

发表于 2017-12-20 分类于 健康 本文字数： 355 阅读时长 ≈ 1 分钟

精油保存： 冰箱冷藏，5年

面部精华油：
精油比例1%-1.5%,约10ml基础油+2-3滴精油

用手掌左右搓揉瓶子，

净痘淡印： 薰衣草精油1滴 + 茶树精油1滴 + 荷荷巴油10ml
使用纯露后，取2-3滴涂于面部，轻柔按摩至吸收，之后再配合乳液，面霜保湿。
也可用茶树或薰衣草单方精油直接点涂痘痘（单方精油刺激性较大，尽量避免，以免皮肤过敏）

茶树精油是被临床医学证实有治疗痤疮效果的天然物质，并且可以直接涂在青春痘上（The Journal of Applied Microbiology, jan. 2000, P170-175）。
但是注意，纯的茶树精油一定要涂在干净并且没水的皮肤上才有最佳效果。
倒几滴茶树油到棉签，化妆棉或纸巾上。轻轻地把茶树精油拍到任何丘疹或痘痘上。稍等几个小时或者等它在痘痘上过夜之后，皮肤的发红和肿胀基本都会缓解。

测量土壤PH值

发表于 2017-12-16 更新于 2017-12-31 分类于 园艺 本文字数： 646 阅读时长 ≈ 1 分钟

用紫甘蓝测量土壤pH

  • 用小刀或料理机把紫甘蓝头捣碎。含紫甘蓝汁的溶液随所接触到的物品pH不同而变成不同的颜色。
  • 把蒸馏水烧开。使用纯蒸馏水会得到更准确的pH测试结果。
  • 把捣碎的紫甘蓝加到烧开的蒸馏水中。让其浸泡十分钟，然后把固体捞出啦，留下紫色的溶液。这个溶液的pH值应该为中性7。
  • 测试紫甘蓝汁。把少量的紫甘蓝汁分到两个杯子中，在一个杯子中加入醋，另一个杯子中加入小苏打。醋是酸性，会使溶液变为艳粉色。小苏打溶液是碱性会使溶液变为蓝色或绿色。
  • 测试土壤。在一个干净的杯子里倒入几英寸深的紫甘蓝汁然后加入一到两勺土壤。等待三十分钟，检查溶液的颜色。
    • 紫红或紫色表示土壤pH接近7，是中性。
    • 粉色表示土壤为酸性，pH在1到7之间。土壤越酸，粉色会越鲜艳。
    • 蓝色或绿色表示土壤为碱性，pH在8到14之间。土壤越碱，绿色会越鲜艳。

用醋和小苏打测量土壤pH

  • 从花园中取一杯土壤。分别舀几勺到两个容器中。
  • 在一个容器中加入醋。如果发出嘶嘶声，表明你的土壤是碱性。如果这种情况，就不需要进行下一步了。
  • 在另一个容器中加入水。水要足以使土壤润湿并形成泥浆。往这个杯子中加入小苏打；如果发出嘶嘶声，表明你的土壤是酸性。
  • 再次检查两个土壤样品。如果都没有发出嘶嘶声，说明你的土壤很有可能是中性pH值为7。这是很好的，是适合大多数植物生长的pH。

改变土壤的pH

  • 减少土壤酸性。如果你的土壤pH小于7，则往土壤上撒些石灰或木灰。这两样东西都能在当地园艺店买到。
  • 减少土壤碱性。如果你的土壤pH大于7，则加些有机物质如：松针，泥炭藓，或枯树叶。
