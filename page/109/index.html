精油疗法-乳香精油

发表于 2017-12-31 分类于 健康 本文字数： 504 阅读时长 ≈ 1 分钟

从乳香树脂中提炼出来，散发着温馨的木质香气和淡淡果香，给人以圣洁，灵动之美。淡化疤痕，对付干燥，老化，暗沉的肌肤，又可用于治疗皮肤溃疡，具有调气活血的功效

精油档案

  • 别名：马尾香，天泽香
  • 英文名：Frankincense
  • 学名：Boswellian carteri
  • 萃取部位：树脂
  • 提取方法：蒸馏法
  • 产地：马达加斯加，科摩罗岛，阿拉伯半岛
  • 特征：无色至淡黄色近黄棕色粘稠状液体
  • 香味：温馨清纯的木质香气，淡淡的果香及柠檬的味道
  • 挥发性：中度挥发

适合搭配的精油

  • 薰衣草；甜橙；香蜂草；广藿香；檀香；天竺葵；葡萄柚；松针；罗勒；黑胡椒

功效

  • 皮肤：淡化疤痕，皱纹，对抗干燥，老化，暗沉的肌肤，收敛功效也常用来治疗青春痘或发炎皮肤
  • 心理：抑制生气，急躁，，沮丧，哀伤等消极情绪
  • 医学：
    • 治疗咳嗽，喉咙痛，支气管炎，呼吸系统各种疾病都有缓解功效
    • 改善生殖泌尿系统，对膀胱炎，尿道炎，月经不调等有一定的疗效
    • 提升免疫力，预防流行性感冒及其他传染病

注意事项

  • 没有特别需要注意的地方，皮肤敏感者使用前先测试一下是否过敏

保存

放在深色玻璃瓶中，避免阳光直射和温度的波动。用完后立即拧紧，避免氧化。
未开封可保存3-5年，已开封最好在1年内用完。调和后2个月内用完最佳。

园艺-芳草种植注意事项

发表于 2017-12-31 分类于 园艺 本文字数： 401 阅读时长 ≈ 1 分钟

购买幼苗的时间

香草在不同季节会呈现不同的状态，因此我们需要在小苗最健康的季节购入。
一般4月中旬-5月中旬是最适合的。每年这段时间会很容易买到各种香草幼苗。

购买幼苗看什么

叶子能体现出香草的全部状态。购买时要上下左右，翻转过来好好检查叶子。
如果幼苗节与节之间很长，而且长的还很高，叶子小，就不好了。
看花盆底部，如果根部露出过长，甚至有些干枯，说明放了很长时间了。我们要选购那些只露出一点儿的幼苗。
即使根没有露出，茎看起来很结实，我们也需要抓住颈部轻轻摇晃一下，看看根部是否牢固
如果购买花苗时，要选择带花骨朵的，越多越好。

倒盆

如果花盆底部露出很多根，或者没露出根，但是生长缓慢，叶子变黄，变小，这些情况一般是停止生长的时候才会出现。
说明需要倒盆了，根部已经长满整个花盆，导致营养不良。
春秋两季适合倒盆，一般倒盆之后要浇透水，放在半阳地几天。
繁殖能力低的香草一般1-3年倒1次盆，繁殖能力强的1年倒1-2次盆。
