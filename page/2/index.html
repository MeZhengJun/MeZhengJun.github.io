园艺-葱

发表于 2022-04-06 分类于 园艺 本文字数： 495 阅读时长 ≈ 1 分钟

超市买来的葱，根部保留一小段，直接就可以种到土壤中

温度：香葱喜凉爽的气候，香葱耐寒、耐热性较强，四季均可种植，发芽适温为1320℃，茎叶生长适宜温度1823℃，根系生长适宜地温1418℃，在气温28℃以上生长速度慢。
光照：香葱是中光植物，不需要强烈的光照，在强光照条件下组织容易老化，纤维增多，品质变差，但光照过弱，有机物积累少，生长也不良，表现为单株叶片细长，叶色淡黄。
水分：香葱叶片表面有蜡质，能减少水分蒸发，但根系无根毛，吸收肥水能力差，需水量比大葱要少，但不耐干旱，适宜土壤湿度为70％80％，适宜空气湿度为60％70％的沙性土壤中种植。因此，在各生长发育期都必须供给适宜的水分，保持土表湿润，但不耐涝
土壤：种植香葱应选择地块平整，土壤应无污染或经无害化处理，排灌方便，耕作层深厚，结构适宜，理化性良好，以砂壤土或壤土为宜，土壤有机质含量应在1.5%以上，PH值在6.07.0之间
养分：香葱是喜钾植物，每生长1万斤分葱须从土壤中吸收纯氮30市斤，氧化钾35斤，五氧化二磷6斤，在粘壤土上栽培产量高，品质好。 香葱不适宜在沙土地块种植，需氮、磷、钾和微量元素均衡供应，不能单一施用氮肥。

园艺-大蒜

发表于 2022-03-13 更新于 2022-04-15 分类于 园艺 本文字数： 277 阅读时长 ≈ 1 分钟

水培大蒜容易腐烂，生发次数少，
土培大蒜长得好，生发的次数多，还能结蒜头，需要肥沃的土壤，蒜瓣间距不能太近

水培大蒜长出根后移植到土壤中，需要添加底肥。需要光照，
完整的大蒜可以掰成一瓣瓣的，然后直接浅埋土壤中4~6厘米深，适当覆土压实。
之后就养在阳光充足的朝南窗台或朝西的窗台上，每天至少要有六个小时的光照，如果光照不足可以用人造灯光补足，也是能够促进正常的生长的。等到大蒜长到12厘米的时候，你就可以适当割下来一些大蒜叶子，用来做调料也是不错的选择。

剪掉几次蒜苗后，会越长越细，一般 3 次没问题，然后就可以翻土，重新种植了。

农家有七葱八蒜的说法，8月份种蒜。
