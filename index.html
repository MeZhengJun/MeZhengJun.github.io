tushare-学习笔记001-环境搭建

Posted on 2019-06-27 | In Language

github链接

windows 环境

  1. 安装 anaconda
    官网

https://repo.anaconda.com/archive/Anaconda3-2019.03-Windows-x86_64.exe

升级 conda 的版本，

  1. 安装 mongodb：注意安装的时候选择自定义，安装过程中去掉 compass 的选项。否则会卡住几个小时都有可能
  2. 创建虚拟环境 stock_tushare. 基于 python3.7 版本创建一个名字为 pyt3_stock_tushare 的环境，同时安装 anaconda

安装完毕后通过 conda env list 查看所有的环境

  1. 激活环境 activate pyt3_stock_tushare，退出环境 deactivate
  2. 安装 tushare pip install tushare

  3. 安装 talib 因为 python 是 64 位的，所以这里也需要 64 位的
    https://www.lfd.uci.edu/~gohlke/pythonlibs/#ta-lib
    下载 TA_Lib‑0.4.17‑cp37‑cp37m‑win_amd64.whl
    执行 pip install TA_Lib‑0.4.17‑cp37‑cp37m‑win_amd64.whl

    下面的方法，实际使用有问题，暂时留着备查。
    http://prdownloads.sourceforge.net/ta-lib/ta-lib-0.4.0-msvc.zip 然后安装 python 库 32位的 pip install TA-Lib
    talib
    下载的ta-lib-0.4.0-msvc.zip 中ta-lib的C库是32位的，需要重新在64位下编译ta-lib库。在此，需要安装Microsoft Visual Studio来实现，我安装的是Microsoft Visual Studio2015。
    运行Visual Studio x86 x64 Cross Tools Command Prompt，接着cd c:\ta-lib\c\make\cdr\win32\msvc，在该目录下执行nmake完成文件的编译
    待执行完毕后，再在该目录下执行 pip install TA-Lib

  4. 安装 pip install pymongo

  5. 打开 pycharm 新建项目，使用 conda 创建环境。

REAL. The value is a floating point value, stored as an 8-byte IEEE floating point number.
TEXT. The value is a text string, stored using the database encoding (UTF-8, UTF-16BE or UTF-16LE).
BLOB. The value is a blob of data, stored exactly as it was input.
